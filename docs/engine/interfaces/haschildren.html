Interface HasChildren

This is a mixin, providing parent/child "awareness" for the event.

Scheduling-wise it implements "aligning" parent events by their child events.

critical

critical: boolean

Inherited from ConstrainedEvent.critical

  • Defined in lib/Engine/data/model/event/ConstrainedEvent.ts:152

A boolean field, indicating whether the event is critical or not. The event is critical if its {@link #getTotalSlack total slack} is zero (or less than zero). This means that if the event is delayed, its successor tasks and the project finish date are delayed as well.
