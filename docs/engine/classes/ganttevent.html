Class GanttEvent

The default class for event, used in Bryntum Gantt.

It is configured from the following mixins, providing the maximum scheduling functionality:

  • HasDateConstraint
  • FixedUnits
  • FixedEffort
  • FixedDuration
  • HasAssignments
  • HasChildren
  • ConstrainedEvent
  • EventMixin
  • HasCalendarMixin
