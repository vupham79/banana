Interface ConstrainedEvent

This is a "constrained event" mixin.

It provides both a generic functionality of constraining the start/end dates of the event using constraint intervals and critical path method related API (earlyStartDate, earlyEndDate, lateStartDate, lateEndDate, totalSlack, critical).

It does not implement any constraining logic itself though, leaving this task to other mixins.
